# heartDiseaseDashboard
TITLE : heart disease dashboard 


DESCRIPTION :
The Heart Disease Dashboard is an interactive data visualization tool designed to analyze and monitor factors related to heart disease. The dashboard helps users quickly understand patterns, trends, and relationships in heart disease data through multiple visual elements.

The dashboard includes slicers, which allow users to filter the data dynamically based on different attributes such as age group, gender, or other health indicators. This interactivity enables users to focus on specific segments of the dataset for deeper analysis.

A line chart is used to display trends and changes in heart disease cases across different categories or over time. This helps in identifying patterns and fluctuations that may indicate increasing or decreasing risk levels here it represents total alive by age group

The stacked bar chart is used to compare multiple variables simultaneously. It shows the distribution of heart disease cases across different groups, making it easier to understand how various factors contribute to the overall data.

A ribbon chart is included to highlight ranking changes among different categories. This visualization helps users see how different factors or groups shift in importance or frequency across the dataset.

Together, these visualizations provide a comprehensive overview of heart disease data, enabling better insights and supporting data-driven decision-making. The dashboard is designed to be user-friendly, interactive, and effective in presenting complex health data in a clear and meaningful way.
